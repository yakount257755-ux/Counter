# မြန်မာ ဂဏာန်းပေါင်းစက် (Myanmar AI Calculator)

ဒါကတော့ React, Tailwind CSS နဲ့ Google Gemini AI နည်းပညာကို အသုံးပြုပြီး ရေးသားထားတဲ့ မြန်မာ ဂဏာန်းပေါင်းစက် Web Application ဖြစ်ပါတယ်။

## ပါဝင်တဲ့ လုပ်ဆောင်ချက်များ (Features)

- **ပုံမှန် တွက်ချက်မှုများ**: အခြေခံ ပေါင်း၊ နှုတ်၊ မြှောက်၊ စား လုပ်ဆောင်နိုင်ခြင်း။
- **မြန်မာဂဏာန်း စနစ်**: အင်္ဂလိပ်ဂဏာန်း (0-9) နှင့် မြန်မာဂဏာန်း (၀-၉) ကို လွယ်ကူစွာ ပြောင်းလဲအသုံးပြုနိုင်ခြင်း။
- **AI အကူအညီ**: Google Gemini AI (Flash Model) ကိုသုံးပြီး သင်္ချာပုစ္ဆာများနှင့် မေးခွန်းများကို မြန်မာလို မေးမြန်းဖြေရှင်းနိုင်ခြင်း။
- **မှတ်တမ်း (History)**: တွက်ချက်ခဲ့သည်များကို ပြန်လည်ကြည့်ရှုနိုင်ပြီး ပြန်လည်အသုံးပြုနိုင်ခြင်း။
- **Dark Mode**: နေ့/ည အလိုက် အလင်း/အမှောင် စနစ် (Dark/Light Theme) ပြောင်းလဲနိုင်ခြင်း။

## နည်းပညာများ (Tech Stack)

- **Frontend**: React (TypeScript)
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **AI**: Google GenAI SDK

## အသုံးပြုပုံ (How to use)

1. **API Key ထည့်သွင်းခြင်း**: AI လုပ်ဆောင်ချက်ရရှိရန် `metadata.json` သို့မဟုတ် Environment Variable တွင် Google AI Studio မှရရှိသော API Key ကို ထည့်သွင်းရန် လိုအပ်ပါသည်။
2. **Installation**: `npm install`
3. **Run Project**: `npm start` (သို့မဟုတ် သက်ဆိုင်ရာ command)
